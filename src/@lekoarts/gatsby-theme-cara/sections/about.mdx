# About

### At SmoothPeaks, we make it possible for anyone—not just developers—to create compelling stories using data. 

We are athletes and developers dedicated to creating simple, clear, and blazing fast data pipelines to organize and interpret information. We aid teams tell stories with data to help people attune to their biology. This includes full-stack development using React, Gatsby, Static Site Generators, Ruby On Rails, GraphQL, CI / CD, Docker and, of course, Python!
