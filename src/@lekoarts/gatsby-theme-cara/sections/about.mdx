
# About

### At SmoothPeaks, we make it possible for anyone—not just developers—to create the tools they need to make better decisions using data and storytelling. 

We are athletes and developers dedicated to creating simple, clear, and blazing fast web apps to organize and interpret information. We aid companies tell stories with data to help people attune to their biology. This includes full-stack development using React, Gatsby, Static Site Generators, Ruby On Rails, GraphQL, CI / CD, Docker and, of course, Python!
